Native IEC 61850 Gateway

ARServer bridges IEC 61850 IED values to Modbus TCP and MQTT.

Connect by IP or open an SCL/CID/SCD file, choose SCADA-ready tags, probe them, then publish a clean gateway map for HMI, dashboard, FAT, or lab use.

GitHub Quick Start

Workflow

  1. Add IED by IP or open SCL.
  2. Discover or import IEC 61850 signals.
  3. Probe selected values.
  4. Map to Modbus TCP and/or MQTT.
  5. Start runtime and connect your HMI/SCADA client.

Runtime view

IEC Object | Value | Timestamp | Quality | Type

Device timestamp and quality are read from IEC 61850 companion attributes when available. ARServer keeps local update time separate.

FAQ

Is ARServer open source?

Yes. It is Apache-2.0.

Can I use IP-only setup?

Yes. Enter the IED IP and MMS port, then run discovery.

Can I use SCL/CID/SCD?

Yes. SCL import gives a deterministic engineering model for tag selection and report planning.

Can FUXA connect?

Yes. Configure FUXA as a Modbus TCP client and point it to the ARServer PC.
